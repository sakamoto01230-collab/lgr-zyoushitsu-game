# 上質接客トレーニング

LAOX 時計事業 スタッフ向けの接客トレーニングゲームです。接客フローに沿って、遊びながら接客の型を反復習得します。

- 単一の HTML ファイル（`index.html`）だけで動作します。外部通信・ログインは不要です。
- 章メニューから各章のミニゲームに挑戦できます（来店対応・敬語・お辞儀・身だしなみ・見送り）。
- 回答ごとに正誤と解説が表示され、章の最後にスコアと習得バッジが出ます。
- ポイントは各章の1回目の挑戦のみ加算されます（進捗は端末内にのみ保存）。
- スマートフォン表示に対応しています。

## 使い方

`index.html` をブラウザで開くだけです。

## 公開ページ

GitHub Pages で公開しています（有効化後にURLが発行されます）。
